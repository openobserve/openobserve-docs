# Integration Icons

This directory contains icons for all integrations displayed on the integrations landing page.

## Icon Sources

Icons should be downloaded from the respective service providers or use generic category icons.

For integrations without specific icons, we use category-based generic icons:
- Cloud: cloud icon
- Database: database icon  
- Messaging: message broker icon
- OS: operating system icon
- Server: server icon
- DevOps: devops/pipeline icon

## Naming Convention

Icons should be named using the integration ID from integrations.json:
- Example: `kubernetes.svg`, `postgresql.svg`, `kafka.svg`

## Format

- Preferred format: SVG
- Fallback format: PNG (square, minimum 128x128px)
- Icons should have transparent backgrounds where applicable
