# Examples for function

## add new field

```
.new_field = "abc"
.
```

## change field value

```
.old_field = "new_value"
.
```

## drop field

```
del(.old_field)
.
```

Playground: [https://playground.vrl.dev/](https://playground.vrl.dev/)
