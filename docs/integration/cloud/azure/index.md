# Azure Monitoring & Cloud Observability Integration

OpenObserve integrates with Microsoft Azure to collect logs and metrics from your Azure subscription and services. Many Azure integrations deploy their infrastructure directly from OpenObserve using ARM templates, so you can set up data collection without leaving the product.

Azure integration tiles show a **Deploy** button for any service that exposes an ARM template, and a **Documentation** button for services that link to setup guides instead.
