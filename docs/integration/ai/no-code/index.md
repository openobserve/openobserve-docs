# No-Code AI Platform Observability Integrations

OpenObserve provides observability for no-code and low-code AI platforms, capturing traces and metrics from visual AI builders, chatbot platforms, and workflow automation tools. Monitor invocation latency, session activity, workflow execution, and errors without modifying the platform's source code.

These integrations wrap platform webhook endpoints or native OTLP export capabilities in OpenTelemetry spans, sending telemetry directly to OpenObserve.
